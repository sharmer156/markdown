环境依赖

1.操作系统 node v0.10.28+ reids ~ 2.依赖 3.其他软硬件要求

部署步骤

  1. 添加系统环境变量 export $PORTAL_VERSION="production" // production, test, dev

  1. npm install //安装node运行环境

  2. gulp build //前端编译

  3. 启动两个配置(已forever为例) eg: forever start app-service.js

     forever start logger-service.js
